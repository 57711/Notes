# 函数

## 箭头函数编译

```js
var a = 0;

const obj1 = {
a: 1,
  getA: () => {
      console.log(this)
   console.log(this.a)
  }
}

// 编译后
var _this = void 0;
var a = 0;
var obj1 = {
  a: 1,
  getA: function getA() {
    console.log(_this);
    console.log(_this.a);
  }
};
```

## 函数的上下文

上下文可以用`apply, call, bind` 来实现, 不需要传参来实现。

```js
function A (){
    const obj = {val: 333}
    B.call(obj)
}

function B (){
    C.call(this)
}
function C (){
    console.log('from C', this.val)
}
A()
```

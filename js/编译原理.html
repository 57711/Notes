# 编译原理

编译就是处理字符串

代码字符串 => 词法分析 => tokens => 语法分析 => ast => 语义分析 => analyzed Ast => 代码生成 => 新代码

Tokens 包含标识符和属性， 例如 machtash Ast 表示代码结构和规则 analyzed ast 类型检查，作用域检查

词 -> 句 -> 语
